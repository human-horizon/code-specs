# Профили

## common

Базовый профиль разработки Human Horizon.
Применяется автоматически к каждому проекту Human Horizon.
Проект, который не объявляет профиль, использует common.

## max

Строгий профиль разработки Human Horizon.
Не применяется по умолчанию.
max включается только когда проект явно объявляет:
Профиль Human Horizon: max

## Правило

common — профиль по умолчанию.
max — по запросу.
Правила max не должны применяться к проекту, если проект явно не включает max.
