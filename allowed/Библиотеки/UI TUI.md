# UI TUI

## Статус

Разрешено для терминальных UI-таргетов Human Horizon.

## Область применения

Этот файл относится к allowed/libraries, потому что определяет разрешённые библиотеки, а не общую архитектуру.

## Стек TUI на Go

### bubbletea

Статус: разрешено.
Назначение:

- интерактивные терминальные UI
- меню
- формы
- панели
- разделённые layout'ы

Обоснование:

- архитектура Model / Update / View в стиле Elm
- зрелая экосистема Charm
- подходит для структурированных TUI-приложений

### charmbracelet/x/ansi

Статус: разрешено.
Назначение:

- парсинг ANSI escape-кодов
- интерпретация терминального вывода
- курсор, SGR, стирание и позиционные последовательности

### creack/pty

Статус: разрешено.
Назначение:

- запуск дочерних процессов внутри псевдотерминальных сессий
- сохранение TTY-поведения и ANSI-вывода

## Правило

Проекты с терминальным таргетом могут использовать этот стек только когда проект явно объявляет терминальный таргет.
Этот файл не разрешает эти библиотеки для несвязанных таргетов.
