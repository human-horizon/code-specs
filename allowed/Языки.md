# Языки

Разрешённые языки программирования.

## TypeScript

Статус: разрешено  
Область применения:

- frontend    
- backend    
- tooling    
- shared logic    
- webview UI    
- runtime adapters

Совместимость:

- browser    
- Node.js    
- toolchain-based окружения

Ограничения:

- не использовать any    
- не отключать проверку типов    
- platform-specific API должны быть изолированы от общего слоя

Обоснование:

- основной язык для общей прикладной логики    
- подходит для shared contracts, tools, sessions и UI

## Go

Статус: разрешено  
Область применения:

- CLI-утилиты    
- TUI-приложения    
- системные инструменты    
- bridge-слой    
- сетевые сервисы    
- компиляторы и интерпретаторы

Совместимость:

- cross-platform native tooling    
- desktop и backend-компоненты

Ограничения:

- не использовать как UI-слой (кроме TUI через bubbletea)    
- platform-specific код должен быть явно изолирован

Обоснование:

- один бинарник без зависимостей — идеален для CLI и TUI    
- отличная стандартная библиотека    
- простая кросс-компиляция    
- удобен для thin runtime/tooling layer
