WebView

# Назначение

Таргет WebView описывает поверхности приложения, которые рендерятся через встроенную browser/webview-среду.

# Разрешённые языки

• TypeScript

# Разрешённые технологии

• Solid для UI, когда это разрешено спецификацией проекта
• Tailwind CSS вместе с design tokens
• Tauri или Capacitor только когда это явно разрешено решением проекта

# Правила

• Таргет WebView должен быть явно объявлен проектом.
• WebView UI не должен напрямую зависеть от нативных API.
• Нативные возможности должны быть доступны через явные bridge/adapters.
• Предположения о browser-среде должны быть задокументированы.
