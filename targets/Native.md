Native

# Назначение

Таргет Native описывает нативные desktop- или mobile-оболочки.

# Разрешённые языки

• Go для runtime, tooling и bridge-слоя
• TypeScript для общих контрактов и UI-логики, когда это явно объявлено

# Разрешённые предположения о среде выполнения

• Нативный код должен быть изолирован за явными платформенными границами.
• Общая логика не должна зависеть от платформенно-специфичных API.

# Правила

• Таргет Native должен быть явно объявлен проектом.
• Код нативной оболочки не заменяет общую прикладную логику.
• Платформенно-специфичный код должен находиться за адаптерами.
• Любой новый нативный фреймворк требует решения перед использованием.
