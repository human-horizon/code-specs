GPUI

# Назначение

Таргет GPUI описывает нативный рендеринг через UI runtime в стиле GPUI.

# Разрешённые языки

• Go для bridge/runtime-слоёв, когда это явно объявлено
• TypeScript для общих контрактов и UI-логики, когда это явно объявлено

# Правила

• Таргет GPUI должен быть явно объявлен проектом.
• GPUI-код не должен протекать в общую логику.
• Предположения, специфичные для рендерера, должны оставаться внутри этого таргета.
• Новые зависимости для рендеринга требуют принятого решения перед использованием.
