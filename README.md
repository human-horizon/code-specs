# code-specs

Спецификации и стандарты разработки Human Horizon.

## Структура

```
code-specs/
├── AI.md                          # Правила работы AI-агентов
├── Profiles.md                    # Профили common / max
├── allowed/                       # Разрешительный список технологий
├── architecture/                  # Архитектурные правила и границы
├── decisions/                     # ADR/архитектурные решения
├── design/                        # Дизайн-система и UI primitives
├── forbidden/                     # Явные запреты
├── guides/                        # Методология, стиль кода, TDD
│   ├── common/                    # Базовые гайды (все профили)
│   └── max/                       # Дополнения для профиля max
├── principles/                    # Философия разработки
└── targets/                       # Screen/runtime таргеты
```

## Принципы

Human Horizon использует **явную модель разрешительного списка**:

- **Всё запрещено, если явно не разрешено.**
- Разрешение появляется только через `allowed/`, `architecture/`, `targets/`, `guides/` или принятое решение в `decisions/`.
- Если нужного решения нет в спецификациях, агент не должен придумывать его сам — он должен остановиться и запросить решение.

## Профили

| Профиль | Описание |
|---------|----------|
| **common** | Базовый профиль. Применяется ко всем проектам по умолчанию. |
| **max** | Строгий профиль. Включается только явно: `Профиль Human Horizon: max` |
