## Автодокументация

### TypeDoc с плагинами

```bash
pnpm add -D typedoc typedoc-plugin-markdown
```

`typedoc.json`:

```json
{
    "entryPoints": ["src/index.ts"],
    "out": "docs",
    "plugin": ["typedoc-plugin-markdown"],
    "markdownNavigationLinks": true,
    "sidebarLinkToAsset": true
}
```

### Генерация в CI/CD

```yaml
- name: Generate Docs
  run: pnpm typedoc
- name: Upload Docs
  uses: actions/upload-artifact@v3
  with:
    name: docs
    path: docs/
```

## OpenAPI Spec

### Генерация из кода

```bash
pnpm add -D @tsoa/cli
```

`tsoa.json`:

```json
{
    "entryFile": "src/index.ts",
    "specVersion": 3,
    "outputDirectory": "./src/generated"
}
```

### Пример

```typescript
/**
 * User controller
 */
@Controller('/users')
export class UserController {
    /**
     * Get user by ID
     */
    @Get('/{id}')
    @Response<NotFoundError>('404', 'User not found')
    async getUser(
        @Path() id: string
    ): Promise<User> {
        // ...
    }
}
```

## Версионирование API

### Semantic Versioning

```text
MAJOR.MINOR.PATCH
  |     |    |
  |     |    +— Bug fixes
  |     +—— New features (backward compatible)
  +——— Breaking changes
```

### Deprecation

```typescript
/**
 * Get user data
 * @deprecated since 2.0.0, use getUserV2
 */
async function getUser(id: string): Promise<User> {
    // ...
}
```
